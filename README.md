# Store Rating Platform

A full-stack web application that allows users to submit ratings (1–5) for stores registered on the platform. Built as a coding challenge submission.

## Tech Stack

- **Frontend:** React (Vite) + Tailwind CSS + React Router + Axios
- **Backend:** Node.js + Express
- **Database:** PostgreSQL
- **Auth:** JWT + bcrypt

## User Roles

| Role | Capabilities |
|---|---|
| **System Administrator** | Add/manage users and stores, view dashboard stats, filter/sort all listings |
| **Normal User** | Sign up, browse/search stores, submit or modify ratings |
| **Store Owner** | View their store's average rating and list of raters |
